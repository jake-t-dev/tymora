# tymora

a simple D&D dice bot

[click here for invite](https://discord.com/oauth2/authorize?client_id=1459139873771491441&permissions=67584&integration_type=0&scope=bot)

## usage

* `!test` - returns "test", uptime check for bot
* `!roll` - rolls the specified number of dice with given modifiers, eg `!roll 2d6+4`
